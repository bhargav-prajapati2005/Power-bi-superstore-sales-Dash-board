# 📊 Superstore Sales Dashboard (Power BI)

## 📌 Project Overview
This project is an interactive **Power BI Sales Dashboard** built using the Superstore dataset. It provides insights into sales performance, profit, customer segments, product categories, and regional trends through interactive visualizations.

---

## 🎯 Objectives

- Analyze overall sales and profit performance.
- Identify top-performing products and categories.
- Compare sales across different regions and states.
- Monitor customer segments and purchasing behavior.
- Create an interactive dashboard for business decision-making.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI Desktop
- Power Query
- Superstore Sales Dataset

---

## 📈 Dashboard Features

- 📌 Total Sales KPI
- 💰 Total Profit KPI
- 📦 Total Orders
- 🌍 Sales by Region
- 🗺️ Sales by State
- 📊 Sales by Category & Sub-Category
- 👥 Sales by Customer Segment
- 📅 Monthly Sales Trend
- 🎛️ Interactive Filters (Slicers)

## 📊 Key Insights

- Technology category generated the highest sales.
- West region contributed the highest revenue.
- Consumer segment accounted for the largest share of sales.
- Monthly sales trend helps identify seasonal demand.
- Interactive filters allow detailed analysis by category, region, and date.

---

## 🚀 How to Use

1. Clone this repository.
2. Open **Superstore Sales Dataset.pbix** in Microsoft Power BI Desktop.
3. Refresh the dataset if needed.
4. Explore the dashboard using the available filters and visuals.
